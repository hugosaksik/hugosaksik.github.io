Art directory

A repository of inspiration

Art directory is an ongoing learning project using the framework meteor. The aim of the project was to build a collection of artworks that inspired me, that one could explore. The interface is composed of a grid of pictures, each one linking to a website with more information on the artwork. A system of tags allows to sort and filter the selection of artworks displayed. The filtering system is not yet completed but as the video below shows, the UI already allows to upload artworks.
